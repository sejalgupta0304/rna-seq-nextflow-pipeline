# rna-seq-nextflow-pipeline
Nextflow-based RNA-seq workflow for mouse lung tumor transcriptome analysis including FastQC, fastp, HISAT2, and featureCounts.

## Pipeline Steps
- FastQC
- fastp
- HISAT2
- SAMtools
- featureCounts

## Input
Paired-end FASTQ files

## Output
- QC Reports
- Trimmed Reads
- Sorted BAM files
- Gene Count Matrix

## Tools Used
- Nextflow
- FastQC
- fastp
- HISAT2
- SAMtools
- featureCounts
